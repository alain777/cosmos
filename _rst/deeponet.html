DeepONet¶

Module for DeepONet model

class DeepONet(nets, output_variables, aggregator='*', reduction='+')[source]¶

Bases: torch.nn.modules.module.Module

The PINA implementation of DeepONet network.

See also

Original reference: Lu, L., Jin, P., Pang, G. et al. Learning nonlinear operators via DeepONet based on the universal approximation theorem of operators. Nat Mach Intell 3, 218–229 (2021). DOI: 10.1038/s42256-021-00302-5

Parameters

  • nets (iterable(torch.nn.Module)) – Internal DeepONet networks (branch and trunk in the original DeepONet).

  • output_variables (list(str)) – the list containing the labels corresponding to the components of the output computed by the model.

  • | callable aggregator (str) – Aggregator to be used to aggregate partial results from the modules in nets. Partial results are aggregated component-wise. See pina.model.deeponet.DeepONet._symbol_functions() for the available default aggregators.

  • | callable reduction (str) – Reduction to be used to reduce the aggregated result of the modules in nets to the desired output dimension. See pina.model.deeponet.DeepONet._symbol_functions for the available default reductions.

Example
>>> branch = FFN(input_variables=['a', 'c'], output_variables=20)
>>> trunk = FFN(input_variables=['b'], output_variables=20)
>>> onet = DeepONet(nets=[trunk, branch], output_variables=output_vars)
DeepONet(
  (trunk_net): FeedForward(
    (extra_features): Sequential()
    (model): Sequential(
    (0): Linear(in_features=1, out_features=20, bias=True)
    (1): Tanh()
    (2): Linear(in_features=20, out_features=20, bias=True)
    (3): Tanh()
    (4): Linear(in_features=20, out_features=20, bias=True)
    )
  )
  (branch_net): FeedForward(
    (extra_features): Sequential()
    (model): Sequential(
    (0): Linear(in_features=2, out_features=20, bias=True)
    (1): Tanh()
    (2): Linear(in_features=20, out_features=20, bias=True)
    (3): Tanh()
    (4): Linear(in_features=20, out_features=20, bias=True)
    )
  )
)
