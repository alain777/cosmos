Plotter¶

Module for plotting.

class Plotter[source]

Bases: object

Implementation of a plotter class, for easy visualizations.

plot(pinn, components=None, fixed_variables={}, method='contourf', res=256, filename=None, **kwargs)[source]

Plot sample of PINN output.

Parameters

  • pinn (PINN) – the PINN object.

  • components (list(str)) – the output variable to plot. If None, all the output variables of the problem are selected. Default value is None.

  • fixed_variables (dict) – a dictionary with all the variables that should be kept fixed during the plot. The keys of the dictionary are the variables name whereas the values are the corresponding values of the variables. Defaults is dict().

  • 'pcolor'} method ({'contourf',) – the matplotlib method to use for plotting the solution. Default is ‘contourf’.

  • res (int) – the resolution, aka the number of points used for plotting in each axis. Default is 256.

  • filename (str) – the file name to save the plot. If None, the plot is shown using the setted matplotlib frontend. Default is None.

plot_loss(pinn, label=None, log_scale=True, filename=None)[source]

Plot the loss function values during traininig.

Parameters

  • pinn (PINN) – the PINN object.

  • label (str) – the label to use in the legend, defaults to None.

  • log_scale (bool) – If True, the y axis is in log scale. Default is True.

  • filename (str) – the file name to save the plot. If None, the plot is not saved. Default is None.

plot_samples(pinn, variables=None)[source]

Plot a sample of solution.

Parameters

  • pinn (PINN) – the PINN object.

  • variables (list(str)) – variables to plot. If None, all variables are plotted. If ‘spatial’, only spatial variables are plotted. If ‘temporal’, only temporal variables are plotted. Defaults to None.
