FeedForward¶

Module for FeedForward model

class FeedForward(input_variables, output_variables, inner_size=20, n_layers=2, func=<class 'torch.nn.modules.activation.Tanh'>, layers=None, extra_features=None, bias=True)[source]¶

Bases: torch.nn.modules.module.Module

The PINA implementation of feedforward network, also refered as multilayer perceptron.

Parameters

  • input_variables (list(str)) – the list containing the labels corresponding to the input components of the model.

  • output_variables (list(str)) – the list containing the labels corresponding to the components of the output computed by the model.

  • inner_size (int) – number of neurons in the hidden layer(s). Default is 20.

  • n_layers (int) – number of hidden layers. Default is 2.

  • func – the activation function to use. If a single torch.nn.Module is passed, this is used as activation function after any layers, except the last one. If a list of Modules is passed, they are used as activation functions at any layers, in order.

  • layers (iterable(int)) – a list containing the number of neurons for any hidden layers. If specified, the parameters n_layers e inner_size are not considered.

  • extra_features (iterable(torch.nn.Module)) – the additional input features to use ad augmented input.

  • bias (bool) – If True the MLP will consider some bias.
